Structure du dossier:
	_ function: contient les fichiers source récupérés du code RclusTool qui contiennent les fonctions utilisées.
	
	_server: contient toutes les fonctions server de chacun des onglets.
	
	_ui: contient toutes les fonctions ui de chacun des onglets.
	
	_www: contient le fichier css.
	
	_global.R: fichier contenant la déclaration de l'environnement RclusTool
	
	_app.R: le fichier à exécuter
